English.
We need to create a character selector. 
Use the CharacterSelector scene. 
Generate button should select and display a random character from the Prefabs folder. 
Play button should take the game to the Game scene, where the selected character should be displayed. 
An important requirement is to separate the logic from the Unity UI. 
All logic should be located in classes that do not inherit from MonoBehavior.

Russian.
Нужно создать выбор персонажа.
Сцена CharacterSelector.
По кнопке Generate должен выбираться и отображаться случайный персонаж из папки Prefabs.
По кнопке Play игра должна переходить на сцену Game где должен быть отображен выбранный персонаж.
Важным требованием должно быть отделение логики от unity ui.
Вся логика должна быть в классах, наследованных не от MonoBehaviour.
